# firebase-react-todo
Firebase React ToDoサンプルアプリ

## ホスティング先
https://react-todo-a5363.firebaseapp.com/

# Firebase初期セットアップ
```
$ npm i --global firebase-tools
$ firebase --version
3.17.7
$ firebase login
$ create-react-app firebase-react-todo
$ cd firebase-react-todo
$ firebase init
```

# モジュール
```
$ npm install firebase --save

```

# ビルド
```
$ npm run build
```

# デプロイ
```
$ firebase deploy
```
